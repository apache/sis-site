Mailing lists

We welcome you to join mailing lists and let us know about your thoughts or ideas about Apache SIS.

Apache SIS User Mailing List

This is where user can ask questions about how to use SIS.

  • Subscribe: user-subscribe@sis.apache.org
  • Post (after subscription): user@sis.apache.org
  • Unsubscribe: user-unsubscribe@sis.apache.org
  • Archives:
    • Apache
    • Markmail

Apache SIS Developer Mailing List

This is where the community hangs out and SIS development happens.

  • Subscribe: dev-subscribe@sis.apache.org
  • Post (after subscription): dev@sis.apache.org
  • Unsubscribe: dev-unsubscribe@sis.apache.org
  • Archives:
    • Apache
    • Markmail

Apache SIS Commit Mailing List

This is where the community receives automated notifications of any changes to the sis code and documentation.

  • Subscribe: commits-subscribe@sis.apache.org
  • Unsubscribe: commmits-unsubscribe@sis.apache.org
  • Archives:
    • Apache
    • Markmail
