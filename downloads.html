Download ZIP/GZ files

Apache SIS 0.2-incubating is now available. See the CHANGES file for more information on the list of updates in this initial release.

  • Source:
    • Apache SIS v0.2-incubating [PGP]

The PGP (Pretty Good Privacy) signatures can be verified using any OpenPGP implementation, for example GPG (GNU Privacy Guard). First download the KEYS file as well as the .asc signature files for the relevant release packages. Make sure you get these files from the main distribution directory, rather than from a mirror. Then verify the signatures using the following:

Using GNU Privacy Guard:

gpg --import KEYS
gpg --verify apache-sis-X.Y.Z.tar.gz.asc

Using PGP version 6:

pgp -ka KEYS
pgp apache-sis-X.Y.Z.tar.gz.asc

Using PGP version 5:

pgpk -a KEYS
pgpv apache-sis-X.Y.Z.tar.gz.asc

Download as a Maven dependency

An easy approach to integrate SIS into a Java project uses the Apache Maven dependency management tool to automatically obtain the required Java Archives (JAR) files from the network. Below are examples of declarations in a pom.xml file for building a project with the SIS core module:

Using SIS releases

<properties>
  <sis.version>0.2-incubating</sis.version>
</properties>

<dependencies>
  <dependency>
    <groupId>org.apache.sis</groupId>
    <artifactId>sis-core</artifactId>
    <version>${sis.version}</version>
  </dependency>
</dependencies>

Using SIS development snapshots

<properties>
  <sis.version>0.4-SNAPSHOT</sis.version>
</properties>

<dependencies>
  <dependency>
    <groupId>org.apache.sis.core</groupId>
    <artifactId>sis-referencing</artifactId>
    <version>${sis.version}</version>
  </dependency>
</dependencies>

<repositories>
  <repository>
    <id>apache.snapshots</id>
    <name>Apache Snapshot Repository</name>
    <url>http://repository.apache.org/snapshots</url>
    <releases>
      <enabled>false</enabled>
    </releases>
  </repository>
</repositories>
